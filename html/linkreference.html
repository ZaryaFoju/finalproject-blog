Link Reference

Lessons Feature(s) Application
Lesson 1 HTML - Base HTML elements and layout
  • The use of elements such as article and aside for basic sectioning, use of hyperlinks, and the creation of a navigation bar and footer.
  • Other elements in HTML, such as the p element, were used intensively on the website.
  • Basic HTML was used all throughout the website, thus the link leads to the home page of the site.
Lesson 2 CSS - Basic color and size styling in the page
  • Used the fontface rule in CSS to add the specified fonts for the website.
  • Color scheme used throughout the website with the the use of CSS's color and background-color properties.
  • Font sizes set with the font-size property.
  • Best shown in the about us page that includes different fonts, colors, and font sizes.
Lesson 3 Design Layouts - Design layouts using CSS and HTML
  • Created a basic layout using elements like div and aside for the basic sectioning. CSS used to move HTML in the needed spots.
  • Used positioning styles to set footer and header toward the bottom and top respectively.
  • Fluid layout created for easy web responsiveness.
  • The essay page is a good example of this with the layout of each blog post along with the sidebar.
Lesson 4 Graphic Design w/CSS - Images and Borders
  • Images were used as background in certain parts of the website such as the background image of the logo that appears in the home page (background-image property).
  • Throughout the website images were placed and resized to fit the page content (see newsletter icons).
  • Fluid layout created for easy web responsiveness.
  • Border used in the reference link page along with the newsletter page.
  • Home page is provided as an example, explore through website to find more.
Lesson 5 Design for Mobile Web - Flexbox and responsive design
  • Navigation bar changes based on the width of one's screen using the media queries (displays a navicon instead).
  • Flexbox used to format pages in a specific way (display: flex for wrapping around the page).
  • Pages are also shifted throughout the website based on the user's device width.
  • The single blog pages show the change in the navigation bar along with the main page itself.
Lesson 6 Tables and Columns
  • Mainly used a table within the link reference page.
  • Using CSS, designed the table with style properties to change the color and positioning.
  • Table border was created where one large border on the outside with a smaller one within the table rows.
  • Used the HTML elements for table, tr, th, and dd for each lesson in the link reference.
Lesson 7 Designing a Webform - Creating a newsletter
  • Created a bastic HTML webform in the newsletter page for the user to submit the email.
  • Identified fields on the webform (name phone, email, etc) and used HTML to identify the required ones.
  • Input validation with HTML, styling with CSS so the user is more easily able to interact with the form (and identify invalid text with images and color changes).
Lesson 8 Enhancing the website with Multimedia
  • Added some audio features for web accessibility for readers who need audio to better understand the text.
  • Designed the audio player to better fit the page, added web controls.
  • Transitions applied to buttons throughout the website.
  • This audio was added to every blog post within the website.
Lesson 9 Enhancing the website with Multimedia
  • A light mode and dark mode feature toggled through buttons adds to web accessibilty.
  • Animated navigator added to the navigation bar throughout the pages for mobile responsiveness.
  • Interactive newletter with the addition of choosing to add a message.
